# Basebound Asset Pack

This is a starter prototype asset pack cropped from the generated Basebound concept sheets.

## Suggested placement
Copy this folder into your repo as:

`/apps/client/public/assets/basebound/`

or:

`/public/assets/basebound/`

## Contents
- `characters/ranger/` - directional idle, walk, and attack frames
- `mobs/` - main, directional, and action images for six enemy mobs
- `buildings/towers/` - archer, ballista, arcane, and watch tower assets
- `manifest.json` - file map for Codex
- `references/` - original full concept sheets

## Codex instruction
Ask Codex to load assets from `/assets/basebound/manifest.json`, create Phaser animations for the ranger, and replace placeholder art with these PNG assets.
